# Number-Guessing-Game
A fun and interactive Number Guessing Game built in Python. The game challenges players to guess a randomly generated number within a given range, with multiple difficulty levels to choose from.

# Features
Easy Mode → Guess between 1–50, unlimited attempts.

Normal Mode → Guess between 1–100, unlimited attempts.

Hard Mode → Guess between 1–1000, 30 attempts.

Insane Mode → Guess between 1–1000, 20 attempts.

Extreme Mode → Guess between 1–10000, 15 attempts.

Input validation ensures players enter valid numbers.

Replay option after each game session.

# Requirements
Python 3.x

Standard libraries: random, time
